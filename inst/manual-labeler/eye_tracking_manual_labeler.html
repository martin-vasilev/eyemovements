Eye-Tracking Manual Labeler

Load CSV gaze samples, navigate long trials, label fixation/saccade/post-saccadic oscillation/unclear segments, and edit existing segments directly.

1) Load data

Expected columns: time, x, y, and optionally trial.

Column mapping

Navigation & zoom

Use the overview navigator, mouse wheel, or keyboard shortcuts.

Trial navigation

If a trial column is provided, the app filters to one trial at a time.

2) Label selected range

Drag to create a selection. If part of the dragged range overlaps an existing segment, the app creates the new segment only from the first undefined sample onward.
Editing tip: click a segment once to activate it. Then drag the large S / E handles shown under the lower plot, or use the edit buttons below. New segments drawn across existing ones are automatically trimmed to the undefined part of the range. Labels cannot overlap: dragged boundaries stop outside neighbouring segments.

Display options

Dragged boundaries can snap to the nearest sample or local velocity change. The main time-series panel now includes a PSO diagnostic band with signed x/y velocity and speed to make ringing easier to spot.

3) Export

Exports include a trial column when trials are present.

Import labels

Expected columns: trial (optional), start_time, end_time, label.
